# Создание и подклюения GITHUB

### Создания репозитория на GitHub
>Просто создал репозиторий.

### Подключение:

* Создание внутреннего Git  
 ```git init```
* Смотри данные пользователя на пк  
```git config user.name```  
```git config user.email```

* Смотри подключения к серверным Git  
```git remout -v```

* Содаю первый файл  
```git add README.md```


* Commit первого файла  
```git commit -m "first commit"```

* Создание первой ветки Main  
```git branch -M main```

* Подключение к GitHub  
```git remote add origin https://github.com/DironWeit/golang-todoapp.git```

* Делаем пуш  
```git push```
  
*# но он не сработал и пишем это для пуша  
```git push --set-upstream origin main```

* Смотрим наши ветки  
```git branch```

* Создание новой ветки infra/env-setup  
```git branch infra/env-setup```

* Смотрим наши ветки  
```git branch```

* Переходим на ветку infra/env-setup  
```git checkout infra/env-setup```
